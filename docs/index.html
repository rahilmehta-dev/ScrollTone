Features
Simple by design.
🎙️

20 Neural Voices

American & British, male & female. Preview any voice before you convert.

📚

Batch Convert

Drop multiple EPUBs at once. Each book gets its own output folder.

🔒

Fully Local

Model weights are bundled in the image. Your books never leave your machine.

🗂️

Per-Chapter Files

Every chapter is its own file. Optionally merge into a single full-length audio.

🎚️

WAV & MP3

Lossless WAV or compressed MP3 with ID3 tags — title, author, cover art.

⚡

Live Progress

Download chapters as they finish. No waiting for the whole book to complete.

Install
One command.
bash 
# Pull the image (~1.5 GB, includes model weights)
docker pull rahilmehta/scrolltone

# Run
docker run -p 7860:7860 rahilmehta/scrolltone

# Optional: save files to your Mac/PC
docker run -p 7860:7860 -v ~/Audiobooks:/out -e OUTPUT_DIR=/out rahilmehta/scrolltone

# Open
open http://localhost:7860
