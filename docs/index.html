Features

Everything you need.
Nothing you don't.

ScrollTone keeps the pipeline simple — drop a book, pick a voice, get audio.

🎙️

20 Neural Voices

American and British accents — male and female — powered by Kokoro-82M, a state-of-the-art open-source TTS model. Preview any voice before converting.

📚

Batch Convert

Drop multiple EPUBs at once. Each book runs as an independent job and output is automatically organized into per-book subfolders named after the title.

🗂️

Per-Chapter Files

Every chapter is saved as an individual audio file so you can jump straight to any part of the book. Optionally merge everything into a single full-length file.

🔒

100% Local

Your books never leave your machine. The Docker image bundles the Kokoro model weights so there are no downloads or API calls at runtime.

🎚️

MP3 + WAV Output

Choose between lossless WAV or compressed MP3 with configurable bitrate. MP3 files include full ID3 tags — title, author, album art, and track number.

⚡

Live Progress Stream

Watch each chapter complete in real time via a live log stream. Download chapters immediately as they finish — no need to wait for the whole book.

Installation

One command to start

ScrollTone ships as a Docker image with everything pre-installed — Python, PyTorch, FFmpeg, and the Kokoro model weights.

1

Install Docker

If you don't already have Docker, download Docker Desktop for Mac or Windows — it takes about two minutes.

2

Pull & Run

Open a terminal and paste the command on the right. Docker will pull the image (~1.5 GB) and start the server. Model weights are bundled — nothing downloads at runtime.

3

Open the UI

Visit http://localhost:7860 in your browser. Drop an EPUB, pick a voice, and start converting.

4

Save outputs to your computer

Mount a local folder so audio files are written directly to your machine — no need to download them from the UI.

bash Terminal 
# Pull the image
docker pull rahilmehta/scrolltone

# Run with default temp output
docker run -p 7860:7860 rahilmehta/scrolltone

# Or mount a folder so files are saved locally
docker run -p 7860:7860 \
  -v ~/Audiobooks:/audiobooks \
  -e OUTPUT_DIR=/audiobooks \
  rahilmehta/scrolltone

# Then open:
open http://localhost:7860
The Kokoro-82M model weights (~330 MB) and all 20 voice previews are baked into the image — the container is ready the moment it starts.
