Features
Simple by design.
🎭

Multi Voice

A local LLM detects who is speaking each line. Sarah gets one voice, James gets another. Consistent throughout the whole book.

📣

Chapter Title Narration

Each chapter begins with a natural pause, then the narrator reads the chapter title aloud before the content starts. Just like a real audiobook.

📑

Real Chapter Names

Titles are pulled directly from the EPUB's table of contents — so you see "Chapter I. Mr. Sherlock Holmes." instead of "Section 1".

📡

Broadcast Audio

One checkbox runs your audio through the same chain used in podcasts and radio. Compression, warmth, harshness cut. Sounds studio ready.

🎙️

20 Neural Voices

American and British, male and female. Preview any voice with one click before you commit. Speed defaults to 0.95× for a natural pace.

📚

Batch Convert

Drop multiple EPUBs at once. Each book gets its own folder. Processed one at a time so your RAM stays stable.

🗂️

Chapter Selection

Pick which chapters to convert before you start. Skip intros and appendices. Character counts shown for each one.

🎚️

WAV and MP3

Export lossless WAV or compressed MP3 with cover art, title, author and track number tags baked in automatically.

🔒

Fully Local

Model weights live inside the Docker image. Your books and audio never leave your machine. No accounts, no rate limits.

Install
One command.
bash 
# Pull the image (~1.5 GB, includes Kokoro model weights)
docker pull rahil18/scrolltone:1.0.1

# Run
docker run -p 7860:7860 rahil18/scrolltone:1.0.1

# Optional: save files directly to your Mac/PC
docker run -p 7860:7860 -v ~/Audiobooks:/out -e OUTPUT_DIR=/out rahil18/scrolltone:1.0.1

# Open in browser
open http://localhost:7860
